# 1.配置git的账号

```bash {.line-numbers}
git config --global user.email xulzee@163.com
git config --global user.name xulzee
```

# 2.把文件添加到版本库
```bash {.line-numbers}
#第一步，把文件添加到仓库
git add autopep8.md
#第二步，把文件提交到仓库
#简单解释一下git commit命令，-m后面输入的是本次提交的说明，可以输入任意内容，当然最好是有意义的，这样你就能从历史记录里方便地找到改动记录。
git commit -m "wrote files"
```
